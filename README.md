Instalar dependências: 
  * Phosphor; 
  * Radix; 
  * Prisma; 
  * ViteJS; 
  * Expo;
  * Fastify;
  * TailwindCSS; 
  * Typescript.

## Server 🖥️
```
  #Iniciando Server
   $ cd server
   $ npm install
   $ npx prisma migrate deploy
   $ npm run dev
```
## Web 💻
```
  #Iniciando Web
   $ cd web
   $ npm install
   $ npm run dev
```
## Mobile 📱
```
  #Iniciando Mobile
   $ cd server
   $ npm install
   $ npx expo start
```
